References

W3Schools

W3Schools is a great website to reference as you are starting to learn web development.
w3schools.com

Mozilla Developer Network

Mozilla Developer Network is the authority resource for in depth and accurate information on web development, it can be less beginner friendly than W3Schools but goes much more in depth with information.
developer.mozilla.org

Stack Overflow

Stack Overflow is where you can find answers to questions in a lot of different coding languages and ask your own questions.
stackoverflow.com

CSS Tricks

Tips, Tricks, and Techniques on using Cascading Style Sheets and much much more.
css-tricks.com

Command Line Cheat Sheet

This is a cheat sheet for command line (unix terminal) commands and a brief description of what they do.
git-tower.com/blog/command-line-cheat-sheet/

Git Cheat Sheet

This is a cheat sheet for git commands and a brief description of what they do.
git-tower.com/blog/git-cheat-sheet/

Can I Use

Not all web browsers have the same functionality and features, this website will help show you what works in different versions of the most widely used web browsers.
caniuse.com

Coolors

Colors are great, color schemes are awesome. Coolors will give you new color schemes with a tap of the space bar, bookmark the URL with the color scheme you like or create an account to save it for later.
coolors.co

Adobe Color

Adobe Color can help you generate your own color scheme by using a color rule or you can explore color schemes others have created already. You do not need to have an Adobe CC membership to use this tool.
color.adobe.com

Google

This is usually where I start my search for answers...
www.google.com

Learn to Code HTML & CSS

This is a nice guide to learning HTML and CSS.
learn.shayhowe.com

webdevfoundations.net/8e/chapters.html

Eloquent JavaScript

This is a book about JavaScript, programming, and the wonders of the digital.
eloquentjavascript.net

Learn Ruby the Hard Way

A free e-book to learn Ruby.
learnrubythehardway.org/book/

Tutorials & Exercises

Lynda

Lynda offers a huge range of tutorial videos on a lot of different software and development languages. Many public libraries across the nation provide free membership to lynda with a library card. To access Lynda.com with a New York Public Library card visit https://www.nypl.org/lynda and click the connect to database link. There are also many free courses you can take at the New York Public Library, checkout https://www.nypl.org/computers for more information.
lynda.com

Udacity

Udacity offers some free online courses that provide some great information.
udacity.com/courses/all

Egghead

Egghead has some great free videos, though some may be a bit advanced for beginners they provide some great information.
egghead.io

Free Code Camp

Free Code Camp has some great lessons to help you learn HTML, CSS, and JavaScript.
www.freecodecamp.com

Codecademy

Codecademy has a variety of lessons to help you learn HTML, CSS, and JavaScript as well as other programming languages.
www.codecademy.com

Khan Academy

Khan Academy may be meant for younger audiences but it breaks things down and explains things in a simple manner that can be easier to follow along with.
khanacademy.org/computing

Flexbox Froggy

Flexbox is the future for creating website layouts with CSS, with 97%+ support worldwide (with prefixes) the future is happening now. This is a game to help you learn flexbox.
flexboxfroggy.com

Flexbox Defense

This is another game to help learn flexbox CSS properties and values.
flexboxdefense.com

HTML & CSS Quiz

This is a basic web app I am developing (still very much in beta) to help students test their HTML and CSS knowledge.
kurtpetrek.com/quiz

YouTube

YouTube has a lot of tutuorial videos and this is where I started and still go to find short tutorials. Some videos are better than others, DevTips and FreeCodeCamp are some good channels to checkout.
youtube.com

codepen.io/

This is a collection of pens I've created on codepen.io to help students learn to code. It contains examples in HTML, CSS and JavaScript.
